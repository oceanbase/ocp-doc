Query OBProxy clusters
===========================================



Description
--------------------------------

You can call this operation to query OBProxy clusters.

Call description
-------------------------------------

### Prerequisite

You have passed the authentication for the OCP application service.

### Request path

`GET /api/v2/obproxy/clusters`

### Request parameters

None

### Response parameters
