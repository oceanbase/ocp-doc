关闭 SQL 采集
==============================

本页面为您介绍关闭 SQL 采集的操作步骤。

操作步骤
-------------------------

1. 登录 OCP 。

   默认进入 **集群概览** 页面。

2. 在 **集群列表** 区域，选择待操作的集群并单击其集群名。

3. 在 **总览** 页面右上角，单击 **关闭 SQL 采集** 。

   ![10141358](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7685987361/p338722.png)

4. 在弹出框中，单击 **确定** 。
