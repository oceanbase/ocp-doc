查询 OCP 服务端的时间 
==================================



**功能说明** 
-----------------------------

查询 OCP 服务端的时间，该接口一般用于确认本地时间与服务端时间是否同步。

**调用说明** 
-----------------------------

**接口约束** 

可以直接调用该接口，无需鉴权。

**请求路径** 

`GET /api/v2/time`

**请求参数** 

无

**返回结果** 

返回服务端时间的文本。

**示例** 
---------------------------

**请求示例** 

```code
GET /api/v2/time
```



**返回示例** 

```code
2020-11-11T15:46:27.316+08:00
```
