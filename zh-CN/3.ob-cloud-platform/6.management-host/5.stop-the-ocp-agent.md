停止 OCP Agent
=================================

本页面为您介绍停止 OCP Agent 的操作。

**操作步骤**
-----------------------------

1. 登录 OCP。



2. 在左侧导航栏中单击 **主机** **。**



3. 单击需要修改的主机操作列的 **...** ，并选择 **停止 OCP Agent** 。

   ![09162133](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/6660562361/p327466.png)


4. 在弹出框中单击 **确定** 。
