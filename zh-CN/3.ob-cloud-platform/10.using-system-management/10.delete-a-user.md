删除用户
=========================

本节介绍如何删除用户。

**前提条件**
-----------------------------

* 当前登录用户拥有系统管理员（ADMIN）或用户管理（USER_MANAGER）角色。



* 待删除的用户不是 admin 。






操作步骤
-------------------------

1. 登录 OCP。



2. 在左侧导航栏中单击 **系统管理** **\>** **安全** 。



3. 在 **用户管理** 页签单击用户列表 **操作** 列的 **删除用户** ，或 **...** 中的 **删除用户** 。

   系统提示"确定要删除用户吗？"。


4. 单击 **删除** 。

   <main id="notice" type='notice'><h4>注意</h4><p>用户删除后不可恢复，请谨慎操作。</p></main>
