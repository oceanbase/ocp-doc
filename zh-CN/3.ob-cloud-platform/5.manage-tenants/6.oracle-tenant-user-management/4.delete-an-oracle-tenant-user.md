删除 Oracle 租户用户 
===================================

本页面为您介绍删除 Oracle 租户用户的操作步骤。

操作步骤 
-------------------------

1. 登录 OCP 云平台，在左导航栏单击 **租户** ，进入租户概览页面。

   

2. 在 **租户列表** 中单击租户名进入 Oracle 租户详情页面。

   

3. 单击左导航栏中的 **用户管理** 。

   

4. 单击需要删除的用户后的 **删除** 。

   

5. 在弹出框中输入 **delete** 。

   ![1](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1825530261/p271201.png)
   

6. 单击 **删除** 。
